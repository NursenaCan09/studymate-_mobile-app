# StudyMate - Mobile Productivity Application

## Project Overview
StudyMate is a mobile productivity application developed as part of the Mobile Technologies course project. The main objective of this application is to help students manage their daily study tasks and improve their focus using the Pomodoro technique.

The application allows users to create, manage, and track their tasks while maintaining productivity through structured study sessions.

## Project Purpose
The purpose of this project is to demonstrate the development process of a functional mobile application including user interface design, local data management, and basic productivity tools.

The application focuses on improving students' time management and study efficiency through task tracking and focus timers.

## Features
- Task creation and management
- Marking tasks as completed
- Pomodoro-based focus timer
- Basic productivity tracking
- Simple and user-friendly interface

## Technologies Used
- **Flutter** – Mobile application development framework
- **Dart** – Programming language used for application logic
- **SQLite** – Local database for storing user tasks
- **Git & GitHub** – Version control and project management

## Application Structure
The application consists of several core screens:

- **Home Screen:** Displays the list of tasks created by the user.
- **Add Task Screen:** Allows users to create and store new tasks.
- **Pomodoro Timer Screen:** Provides a focus timer based on the Pomodoro technique.
- **Statistics Screen:** Displays simple productivity information.

## Data Management
User-generated task data is stored locally using SQLite. This ensures that the data persists even after the application is closed and reopened.

## Educational Context
This project was developed as a course requirement for the *Mobile Technologies* program. The goal is to apply fundamental mobile development concepts such as user interface design, data persistence, and version control practices.

## Future Improvements
Possible future enhancements include:

- Push notification reminders
